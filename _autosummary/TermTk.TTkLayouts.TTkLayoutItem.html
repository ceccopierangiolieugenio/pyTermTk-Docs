TTkLayoutItem is the base class of layout Items inherited by TTkLayout, TTkWidgetItem, and all the derived layout managers.

Parameters:

  • row (int) ‚Äì (used only in the TTkGridLayout), the row of the grid, optional, defaults to None

  • col (int) ‚Äì (used only in the TTkGridLayout), the col of the grid, optional, defaults to None

  • rowspan (int) ‚Äì (used only in the TTkGridLayout), the rows used by this, optional, defaults to 1

  • colspan (int) ‚Äì (used only in the TTkGridLayout), the cols used by this, optional, defaults to 1

  • layoutItemType (TTkConstant.LayoutItemTypes) ‚Äì The Type of this class, optional, defaults to TTkK.NONE

  • alignment (TTkConstant.Alignment) ‚Äì The alignment of this item in the layout (not yet used)
